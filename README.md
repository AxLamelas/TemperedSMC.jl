# TemperedSMC.jl

Implementation of traditional and waste-free [Dau and Chopin](https://doi.org/10.1111/rssb.12475) sequential monte carlo .
Some adaptive MCMC kernels are implemented like [delayed rejection](https://doi.org/10.1093/biomet/88.4.1035) and [auto step](https://arxiv.org/abs/2410.18929).
